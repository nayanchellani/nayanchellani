## 🔭 Featured Projects

### 🦈 [PitchZone](https://pitch-zone.vercel.app/) — Virtual Shark Tank Platform
A full-stack MERN application enabling structured interactions between **entrepreneurs and investors**.

* **Security:** JWT-based authentication with protected routes & Role-Based Access Control (RBAC).
* **Architecture:** Modular REST APIs and reusable React components.
* **Stack:** MongoDB · Express · React · Node.js

🟢 **Status:** Live Deployment
